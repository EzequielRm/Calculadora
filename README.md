# Calculadora

Calculadora web simple para sumar, restar, multiplicar y dividir. Usa `index.html`, `index.js` y `style.css`.

Demo: https://ezequielrm.github.io/Calculadora/

Instrucciones:

- Abrir `index.html` en el navegador para usar la calculadora.
- O ejecutar un servidor local desde la carpeta del proyecto.

Archivos incluidos:

- `index.html`
- `index.js`
- `style.css`
